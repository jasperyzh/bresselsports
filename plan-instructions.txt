## ⚠️ REMAINING TASK: Fluent Forms Setup

### Why This Needs Manual WP Admin

Fluent Forms requires UI interaction to:
1. Create form fields (drag & drop builder)
2. Configure email routing
3. Set up form notifications
4. Configure AJAX settings

These cannot be scripted via WP-CLI or database manipulation without complex SQL queries.

### Instructions for Manual Setup:

1. **Login to WordPress Admin:** `http://localhost:8080/wp-admin/`

2. **Create Contact Form (ID: 1)**
   - Go to: Fluent Forms → Add New Form
   - Name: "Contact Form"
   - Add fields: Name (text), Email (email), Message (textarea), Intent (hidden field with default value "general")
   - Save & copy shortcode: `[fluentform id="1"]`

3. **Create Newsletter Form (ID: 2)**
   - Go to: Fluent Forms → Add New Form
   - Name: "Newsletter Signup"
   - Add field: Email (email)
   - Save & copy shortcode: `[fluentform id="2"]`

4. **Update Contact Page Content**
   - Go to: Pages → Contact → Edit
   - Add shortcode: `[fluentform id="1"]`
   - Save
