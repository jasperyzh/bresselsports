# BRESSEL™ WordPress Framework

A high-performance Padel Academy website built with a modern Vite + Tailwind v4 stack.

## 🚀 Quick Start

1. **Environment**: `docker compose up -d`
2. **Theme Dev**: `cd twentytwentyfour-bressel && npm run dev`
3. **Scaffold Content**: `./setup-wp-content.sh`

## 📁 Project Map

- `/twentytwentyfour-bressel`: The custom child theme (Core logic, styles, templates).
- `/docs`: Technical specifications, deployment plans, and setup guides.
- `AGENTS.md`: Critical guidelines for AI agents working on this codebase.
- `docker-compose.yml`: Infrastructure configuration.

## 🛠 Tech Stack

- **PHP 8.2+** | **Tailwind CSS v4** | **Vanilla JS** | **Vite 6**
- **CMS**: WordPress (Twenty Twenty-Four Child Theme)
- **Fields**: CMB2 | **Forms**: Fluent Forms
